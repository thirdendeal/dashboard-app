# Dashboard-App

Dashboard de uso interno para cotação de produtos com fornecedores relacionados.

Empresa fictícia Burguer Burguer.

## Setup

1. Instalar PHP (7.4 ou superior)
2. Instalar MySQL
3. Criar usuário MySQL `dashboard` (senha `development-password`)

## Inicialização

1.  Iniciar servidor MySQL (porta padrão 3306)
2.  Iniciar servidor PHP:

        php -S localhost:8000
